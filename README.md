# MYA — Web Developer Portfolio

## ✨ Features

- Fully responsive — mobile, tablet, and desktop
- Fixed header with animated mobile hamburger menu
- Smooth scroll navigation
- Editorial-style hero section with oversized typographic watermark
- Projects grid with hover interactions
- Contact form with client-side validation
- Clean footer navigation
